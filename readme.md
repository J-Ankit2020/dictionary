## Requirements

- Node.js
- npm

## Features

- Light/dark mode toggle
- Search word definition with instant results
- Word pronunciation
- Word usage in sentences



## Run Locally

Clone the project

```bash
  git clone https://github.com/J-Ankit2020/dictionary.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
